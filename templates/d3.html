ABOUT IT

Features

  • Mouse hovering revealing data for each data point.
  • Plot zooming in and out with dynamic axis labelling.
  • Addition and/or removal of data by user, with automated plot scale adjustment.
  • User can directly select a node from graph to delete.
  • The size pf each node increases with increase in the RMS value.
  • Original-Button to reload the data from source location.
  • Since it is deployed on heroku server, so changes made at one place will be reflected on all devices viewing it.

Function

  • Data : The box shows reveals the data for each data point on hover.
  • Add : A user need to fill the text to add the data.(Period,Period Derivative,RMS required)
  • Delete : The user can input the detail of the data point or hover to select a data point to delete.
  • Original : Button to reload the data from source location.
